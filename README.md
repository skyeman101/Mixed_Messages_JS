# Mixed Messages JavaScript Portfolio Project

### Project goals:

* Build a message generator program using JavaScript.
* Use Git Version control
* use command line.
* Develop locally on your computer.

### Technologies:

* VS Code
* JavaScript
* Command Line
* Git and GitHub

To run this project:

```
node mixed_messages.js
```
